# Stork · папка фотографий

## Что это

15 фотографий, которые приложение Stork использует на главной, в каталоге, корзине и других экранах. До того как мы наймём фуд-фотографа на продакшн, эти фотки нужны для демо-показа клиенту — чтобы он увидел финальный визуал.

## Как добавить

**Вариант 1 — через ChatGPT (рекомендую)**
1. Открой ChatGPT с подключённым DALL-E (или любой генератор изображений).
2. Скопируй промпт под каждой фоткой ниже, вставь в чат, дождись результата.
3. Сохрани как `имя.jpg` в эту папку (`photos/`).
4. Открой любой HTML — фото подтянутся автоматически.

**Вариант 2 — найти на стоках вручную**
Pexels.com, Pixabay.com, Unsplash.com — бесплатные стоки. Поиск по описанию фотки. Скачиваешь, переименовываешь, кладёшь в папку.

**Вариант 3 — снять самим**
Если у клиента есть товары в наличии, можно за час нащёлкать на телефон с верхней подсветкой и белой простынёй. Не идеально, но для демо хватит.
