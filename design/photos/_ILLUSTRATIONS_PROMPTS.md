## РЕКОМЕНДАЦИИ ПО ГЕНЕРАЦИИ

**Для ChatGPT (DALL-E 3):**
- Вставляй промпт целиком
- Если выдаёт не то — добавь в конце: `Make sure NO text appears anywhere. Strict color palette. Simple flat vector style.`
- Запроси сразу 2-3 варианта через `Generate 3 different versions`

**Для Midjourney:**
- Добавь в конец `--ar 1:1 --style raw --v 6`
- Для большей чистоты вектора добавь `flat design, vector art, minimal, clean shapes`

**Для консистентности всех иллюстраций между собой:**
- Сначала сгенерируй ОНБОРДИНГ 1 (1.1) — закрепи стиль
- Скажи модели: «Use the same illustration style as the previous one» для каждой следующей
- Если стиль уезжает — попроси перегенерить с явной отсылкой к палитре

**Куда положить когда сгенеришь:**
- `./photos/onboarding-delivery.jpg` (для 1.1)
